## 📥 Fontes de Dados

### Bitcoin Price API

-   Origem: API pública (ex: CoinGecko)
-   Formato: JSON
-   Frequência: até 2h
-   Tipo: streaming simulado + batch

### Câmbio USD/BRL

-   Origem: API financeira
-   Frequência: diária
-   Tipo: batch
* * *

## 📦 Classificação

### Dados Operacionais (Batch)

-   histórico diário de preços
-   taxa de câmbio

### Dados de Streaming (simulado)

-   preços recentes em intervalos de 2h e 4h
